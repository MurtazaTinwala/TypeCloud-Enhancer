# CDP Keyboard Shortcuts

* Alt + Q	-------- Clear formatting
* Alt + B	-------- Bullets
* Alt + N	-------- Numbered list
* Alt + U	-------- Edit/Insert link
* Alt + I	-------- Remove link
* Alt + C	-------- Code
* Alt + S	-------- Scree text
* Alt + M	-------- Set start of numbered list
* Alt + P	-------- Page break
* Alt + 1	-------- Heading level 1
* Alt + 2	-------- Heading level 2
* Alt + 3	-------- Heading level 3
* Alt + 4	-------- Heading level 4
* Alt + 5	-------- Paragraph
